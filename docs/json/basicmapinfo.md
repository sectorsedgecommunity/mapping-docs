# Basic map info
Allegiance is optional, and doesn't show up in map vote list.

```json
// Allegiance: "aegis", "corahk", "devoid", "helix", "irridyne", "nova", "soltec", "terra"
"Name": "Map name",
"Allegiance": "",
"Description": "Map description\nCreated by: map author(s)",
"MapSize": [ 256, 96, 768 ],\
```
